Sintaxis & Reglas

Para poder empezar a realizar un código es importante saber las partes en la que se componen. Primero se analiza la primera parte.


Una vez creado el archivo se escribe DOCTYPE donde aparecera todo lo que aparece en la imagen.

  • DODCTYPE.- La función de DOCTYPE es especificarque tipo de lenguaje se va a utilizar en el documento.
  • HEAD.- Puede contener otros tags de HTML que brindan información como el titulo, descripción, palabras clave, etc.
  • TITLE.- Es utilizado para declarar el nombre del documento HTML.
  • META.- Se usa para declarar metada.Puede incluir descripción, palabras clave, etc. También para hacer refresh y agregar cookies.
  • SCRIPT.- Se usa para declarar el script (como JavaScript) dentro del documento HTML.
  • LINK.- Es usado para definir un link de una fuente externa.

Para insertar diseños mediante una ventana CSS, la línea 9 en href simplemente se escribe el nombre de la ventana CSS para que directamente desde la otra ventana se actualicen los cambios.

HTML

HTML (HyperText Markup Language) es el lenguaje para paginas web. Te permite crear una pagina web, agregar contenido y colocarlo en donde tu quieras.(casi todo alguno depende más de css). Los tags en Html tiene un significado literal de las palabras por lo que no es muy dificil averiguar su función.

En esta tabla mostrare un poco de las etiquetas y elementos que hay en HTML, estos pueden tener tag de comienzo y de final (abre y cierra).

Elemento Descripción
h1 Este elemento funciona para hacer titulos, va de h1 hasta h6.
p Este elemento como el nombre lo dice agrega un parrafo a la página.
br Este elemento no tiene cierre, pero su función es hacer un salto de línea
b / strong Sus funciones son las misma y hacen que la letra sea en negritas.
i / em Sus funciones son las mismas y lo que hacen es convertir la letra en un estilo cursiva
mark Su función es subrayar la palabra como si fuera marcatexto.
small Su función es hacer la letra pequeña cuando se encuentra entre los tags.
del Esta etiqueta elimina indirectamente lo que se quiere decir.
ins Esta etiqueta subraya el texto

CSS

CSS (Cascading Style Sheets) este modo es el encargado de, como dice el nombre, dar el estilo a la página, puede dar desde color situar la posición de las cosas (imagenes o texto), hasta hacer una pequeña animación.

La sintaxis para agregar estilo a la pagina es muy sencillo ya que simplemente tienes que escribir lo que quieres modificar, pero esto se basa por modulos. Aquí una imagen de como sería la sintaxis:

  • Se selecciona el punto de HTML que se quiere modificar.
  • Los bloques contienen una o más declaraciones separadas por semicolumnas.
  • Cada declaración incluye una propiedad, nombre y valor separada por columnas.
  • La declaración termina con una semi columna, y el bloque de la declaración está rodeado por llaves.

Existen muchas propiedad y elementos tanto en HTML y CSS algunas son iguales y otras no, pero si sabes para que funcionan no habra problema.


Realizar tablas en HTML

El cuerpo de las tablas es muy similar a un código en HTML, ya que ambas tienen un head y body, en este caso, head serían las columnas y el body el contenido.

La estructura y elementos que pueden incluir una tabla son:

  • TABLE.- Crear tabla
  • CAPTION.- Nombre de la tabla
  • THEAD.- Encabezado de la tabla
  • TBODY.- Cuerpo de la tabla
  • TH.- Número de columnas
  • TR.- Renglones
  • TD.- Fila
  • COLSPAN.- Combina columnas
  • ROWSPAN.- Combina filas

Formulario











Este form contiene otros atributos que pueden ser cajas de texto, rango, contraseña, ingresar un url, seleccionar un archivo, combobox, e incluso un cuadro de texto modificable. Todo se maneja con la misma sintaxis empezando con un Input tipo Type.

Su opción llamada "placeholder" permite agregar texto por default dentro de la caja, sea cual sea su atributo. Tambien se pueden agregar checkbox, radiobutons y botones.

Estos formularios son enfocados más en el llenado de datos. Registrar una persona vía web o crear sitios de ese tipo. Algunos atributos que se le puede dar a las cajas de texto es impedir la escritura en el mismo, aplicando un valor llamado readonly su mismo nombre lo dice solo es de lectura, o, un disabled el cual no permite la escritura y está bloqueado, se le puede dar un size a la caja de texto, maxlenght que es el número maximo de caracteres permitidos. Incluso un autocompletar para rellenar con valor ya predefinidos previamente.
